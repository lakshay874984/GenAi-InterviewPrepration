# 🤖 GenAI Resume & Interview Platform

AI-powered platform that analyzes resumes, compares them with job descriptions, and generates personalized interview questions and preparation plans.

---

## 🔹 Overview
This is a full-stack GenAI-based job preparation platform designed to help users improve their chances of getting hired. Users can upload their resumes and provide a job description, and the system uses AI to analyze compatibility, identify skill gaps, and generate tailored interview questions along with a preparation roadmap.

## 🛠 Tech Stack

**Frontend**
- React.js
- SCSS
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB Atlas

**AI Integration**
- Google Gemini AI API

**Other Tools**
- JWT Authentication
- Multer (File Uploads)
- Puppeteer (PDF Generation)

---

## ✨ Features

- 📄 Resume upload and job description input  
- 🤖 AI-powered resume analysis using Gemini API  
- 📊 Skill gap identification  
- 🎯 Personalized technical & behavioral interview questions  
- 🗺 Custom preparation roadmap  
- 📑 ATS-optimized resume generation (PDF)  
- 🔐 Secure authentication using JWT
